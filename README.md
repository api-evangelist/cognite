# Cognite (cognite)

Cognite is an industrial AI and data platform provider whose core product, Cognite Data Fusion (CDF), contextualizes and integrates data from OT, IT, and ET systems across oil and gas, energy, manufacturing, and mining industries. CDF exposes a comprehensive REST API covering assets, time series, events, files, 3D models, data modeling, entity matching, transformations, functions, and workflows, with SDKs available for Python, JavaScript, Java, .NET, and Rust.

## APIs

### Cognite Data Fusion API

The Cognite Data Fusion (CDF) REST API provides programmatic access to industrial data including assets, time series, events, files, sequences, 3D models, data modeling spaces and instances, entity matching for contextualization, SQL transformations, serverless functions, workflow orchestration, and SAP writeback. Supports OAuth 2.0 and API key authentication against project-scoped endpoints.

- **Human URL:** [https://api-docs.cognite.com/](https://api-docs.cognite.com/)
- **Base URL:** `https://api.cognitedata.com`
